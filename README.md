# 🪷 Kalavidara-Balaga
### Folk Artist Talent Hub — Karnataka

A professional Android marketplace connecting traditional Karnataka folk artists
(Dollu Kunitha, Pooja Kunitha, Yakshagana, Kamsale) with event organizers and customers.

## 🎯 Problem Statement
Local folk troupes depend on seasonal events with no digital presence.
This app gives them a searchable profile, portfolio, and booking system.

## 📱 Features
- Artist/Troupe profiles with portfolio gallery
- Search by District and Art Form
- Booking inquiry form
- Direct call via Android Intent
- Cultural Impact Goals section

## 🛠️ Tech Stack
- Kotlin + Jetpack Compose
- Jetpack Navigation Component
- Firebase Firestore
- Material Design 3
- Coil (image loading)
